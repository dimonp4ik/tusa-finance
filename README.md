# Tusa Finance

Инвест-бот на недели. Ищет дивидендные акции и активы на "иксы" (акции + крипта OKX EU), шлёт сигналы в Telegram. Без автоторговли пока — только сигналы.

## Что нужно, чтобы запустить (2 ключа, оба бесплатные)

### 1. Telegram-бот
1. В Telegram напиши [@BotFather](https://t.me/BotFather) → `/newbot` → следуй инструкциям → получишь токен вида `123456789:AAxxxxxx...`
2. Напиши своему новому боту любое сообщение (просто "привет")
3. Открой в браузере: `https://api.telegram.org/bot<ТВОЙ_ТОКЕН>/getUpdates` → найди `"chat":{"id":ЦИФРЫ` — это твой `TELEGRAM_CHAT_ID`

### 2. Деплой на Railway
1. Railway → New Project → Deploy from GitHub repo → выбрать `tusa-finance`
2. В Variables добавить:
   - `TELEGRAM_TOKEN` = токен из шага 1
   - `TELEGRAM_CHAT_ID` = chat_id из шага 1 (для группы нужен формат с `-100`, не просто `-`; см. .env.example)
   - `ADMIN_USER_IDS` — необязательно, супер-админ (671071896) уже зашит в коде
3. Всё остальное (данные по акциям/крипте) работает бесплатно без ключей — Railway сам подхватит `Procfile` и запустит.

Два независимых цикла:
- **Дивиденды** — раз в сутки (07:00 UTC, `SCAN_HOUR_UTC`) — фундаментал не меняется внутри дня
- **Моментум + аномальный объём** — каждые 4 часа (`MOMENTUM_SCAN_INTERVAL_HOURS`) — тут свежесть важна, особенно для аномального объёма (ловит движение в день старта)

## Кнопки в Telegram
Пришли боту `/start` — появится меню внизу:
- 💰 Дивиденды / 🚀 Моментум / 🔥 Аномальный объём — последний найденный список по запросу (не ждать до следующего скана)
- 🕐 Рынок США — открыт/закрыт сейчас
- 🔄 Запустить скан сейчас — только для админов

## Как это работает
- **Dividend screener** — акции США (весь листинг NASDAQ+NYSE), фильтр по дивдоходности/payout ratio/стажу выплат
- **Momentum screener** — акции + ВСЕ монеты OKX EU, фильтр по росту цены/объёму/RSI за последние недели. Требует, чтобы рост уже случился (+15%/4нед)
- **Unusual volume screener** — акции США, ловит момент СЕГОДНЯ (объём резко выше среднего + цена реагирует) — раньше momentum screener, который видит движение только когда оно уже разогналось
- **Insider buying** — если по кандидату (акция) недавно был открытый рыночный выкуп инсайдером (топ-менеджер/директор купил акции за свои деньги — SEC EDGAR Form 4, официально и бесплатно), это добавляется тегом 💼 к сообщению
- Крипто-кандидаты помечаются "можно с плечом 2-5x" или "только spot" — решает скринер по силе моментума, плечо не навязывается. Низколиквидные монеты помечаются 🎰 (лотерейный тикет, маленький размер позиции)
- Акции — всегда без плеча, только покупка
- Моментум-дайджест — общий топ-5 (акции+крипта вместе, минимум по одной из каждого класса когда есть)

## Контекст рынка в каждом дайджесте (бесплатно, без ключей)
- **Режим рынка США** — SPY против 200-дневной средней. Моментум исторически зарабатывает в бычьем режиме и сливает в медвежьем — в медвежьем дайджест предупреждает
- **Crypto Fear & Greed** (alternative.me) — на Extreme Greed погоня за пампами опаснее, на Extreme Fear исторически неплохая зона набора
- **Funding rate OKX** — если у монеты с рекомендацией плеча перегрет фандинг (лонги переполнены), рекомендация автоматически понижается до "только spot" с пометкой 🌡

## AI-проверка (опционально)
Каждый кандидат получает вторую оценку: 🤖 BUY/SKIP + причина. Провайдер по ключу:
- `CLAUDE_API_KEY` (основной — тот же ключ что у других tusa-ботов). Модель — `claude-haiku-4-5` по умолчанию (~$0.5/мес), `JUDGE_MODEL=claude-sonnet-5` для более умной оценки (~$1.5/мес)
- `DEEPSEEK_API_KEY` — запасной вариант, если Claude-ключа нет

Вердикты пишутся в БД, чтобы потом сверить их точность с реальными исходами — как это делалось с Claude в крипто-боте. Без ключей шаг просто пропускается.

## Бектест (backtest_momentum.py)
Запускается вручную (`python backtest_momentum.py`), не часть живого бота. Проверяет: если бы momentum screener сработал в прошлом, продолжался ли рост дальше?
- **Акции**: рабочий edge, +3пп средний форвард-возврат против бейзлайна (просто держать актив 4 недели в случайный момент)
- **Крипта**: edge около нуля/слегка отрицательный даже на топ-ликвидных монетах — хайповые монеты (памп-дамп) портят статистику. Крипто-momentum сообщения помечены в Telegram ⚠️ как экспериментальные
